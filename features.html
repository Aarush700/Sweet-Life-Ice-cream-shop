Implemented Features

  • Highlighting the menu item in the navigator

    Highlighting the menu item in the navigator corresponding to the current webpage. (Implemented in the JavaScript code of the navigation menu.)

  • Automatic Billing Address

    In page "order.html", a checkbox with the label "Billing address is same as the delivery address" is added somewhere near the field of billing address. When this checkbox is checked, the fields for the billing address will be filled with the delivery address automatically and billing address will disappear. If the delivery address is not completed when this checkbox is checked, display an alert: "Please enter your delivery address first". (Implemented using JavaScript in the "order.html" page.)

  • Multiple Credit Card Options

    Allow users to choose from three types of credit cards: Visa, MasterCard and American Express. Based on the type of credit card, limiting the length of the credit card number to 16 digits for Visa and MasterCard and 15 digits for American Express. (Implemented using JavaScript in the "order.html" page.)
